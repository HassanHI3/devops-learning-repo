# Networking

This section covers core networking concepts required for DevOps, cloud, and infrastructure roles.
The focus is on understanding how systems communicate over a network.

## Topics Covered
- OSI model
- TCP/IP
- IP addressing
- Subnetting and CIDR
- DNS
- Ports and protocols

## Goal
- Understand how data moves across networks
- Build confidence with networking concepts used in cloud environments

- ## Labs

- **Domain + EC2 + DNS (NGINX)**
  - Deployed an EC2 instance running NGINX
  - Opened required ports (SSH 22, HTTP 80)
  - Connected a custom domain using DNS A records
  - Verified access via both public IP and domain name
